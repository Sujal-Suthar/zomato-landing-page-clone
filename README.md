# Zomato Landing Page Clone:-

A responsive Zomato Landing Page Clone built using HTML, CSS, and JavaScript. This project recreates the modern look and feel of the Zomato homepage with a clean UI and responsive design.

## Features:-

- Responsive layout
- Modern UI inspired by Zomato
- Hero section with background video
- Search section
- Food categories
- Popular collections
- App download section
- Footer with useful links
- Smooth user experience

## Built With:-

- HTML5
- CSS3
- JavaScript

## Project Structure:-

```
Zomato-Landing-Page-Clone/
```

You can view the project by opening **index.html** in your browser.

This project was created to practice frontend web development, improve HTML structure, CSS styling, responsive layouts, and JavaScript fundamentals.
